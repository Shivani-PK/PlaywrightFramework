This repository contains Playwright Test Automation framework with Cucumber BDD approach on top over it. This project has been developed using JavaScript.

End-To-End Tests with Page Object Model design pattern can be found here - tests/

Page Objects of Mocha Framework can be found here - pageobjects/

Test data can be found here - testdata/

Reusable API util files can be found here - utils/

Cucumber feature files - features/

Cucumber Step definition files - features/step_definitions/

Cucumber Hook functions - features/support
